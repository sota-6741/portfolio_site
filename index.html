About me

2006年三重県生まれ。高校時代にPythonでツールを作ったことをきっかけに、プログラミングの楽しさに夢中になりました。現在は大学2年生として地中レーダを使った研究に取り組み、日々コードと向き合っています。自然に囲まれた京都・福知山で暮らしながら、地元の子どもたちにプログラミングを教える活動もしています。ものづくりの楽しさと人の役に立てることの喜びを大切に、自身の培ってきた技術で世の中の「ちょっと便利」や「ワクワク」を増やしていきたいです。

Works

PC画面上の文字を自動で検出し、日本語に翻訳するデスクトップアプリです。 ユーザーが翻訳したい範囲を選ぶだけで、OCRによる文字認識から翻訳、結果表示までを一括で行います。 PyQt6とTesseractを組み合わせて開発し、Windows・Linuxの両環境で動作します。

Git hubで詳細を見る

UnityとAR Foundationを用いて制作したAndroid向けのARアプリです。 カメラで水平面を検出し、地面をタップすると花畑が広がります。 ポアソンディスクサンプリングを採用することで、花同士の間隔を保ちながら自然な配置を実現しました。 ARを通して、身近な空間でちょっとした遊び心を楽しめます。

Git hubで詳細を見る

GSAPとSwiperを用いて、ユーザーに心地よい驚きを与えるアニメーション表現を実装したポートフォリオサイトです。 GSAPによるオープニング演出やスクロール連動の文字アニメーション、Swiperによるカルーセルなどを独自モジュールとして設計し、体験全体を統一的に制御しています。
